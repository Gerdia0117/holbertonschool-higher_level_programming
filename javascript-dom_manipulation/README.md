# JavaScript - DOM Manipulation

This project focuses on DOM manipulation using JavaScript, including selecting elements, modifying styles, updating content, and making API requests.

## Learning Objectives

- How to select HTML elements in JavaScript
- Differences between ID, class and tag name selectors
- How to modify an HTML element style
- How to get and update an HTML element content
- How to modify the DOM
- How to make requests with Fetch API
- How to listen/bind to DOM events
- How to listen/bind to user events

## Requirements

- Chrome browser (version 57.0 or later)
- Code must be semistandard compliant
- No use of `var`
- HTML should not reload for each action

## Files

- `0-script.js` - Updates header text color to red
- `1-script.js` - Updates header color on click event
- `2-script.js` - Adds CSS class on click
- `3-script.js` - Toggles between two classes
- `4-script.js` - Adds list items dynamically
- `5-script.js` - Updates header text content
- `6-script.js` - Fetches Star Wars character data
- `7-script.js` - Fetches and lists Star Wars movies
- `8-script.js` - Fetches and displays translation
