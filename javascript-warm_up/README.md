# JavaScript - Warm up

This project is an introduction to JavaScript programming, covering basic concepts and scripting fundamentals.

## Learning Objectives

By the end of this project, you should be able to explain:

- Why JavaScript programming is amazing
- How to run a JavaScript script
- How to create variables and constants
- Differences between `var`, `const`, and `let`
- JavaScript data types
- Control flow statements (`if`, `while`, `for`)
- Functions and their usage
- Variable scope
- Arithmetic operators
- Object and array manipulation
- File importing

## Requirements

- **Editors**: vi, vim, emacs
- **Environment**: Ubuntu 20.04 LTS with Node.js 14.x
- **Code Style**: Semistandard compliant (Standard + semicolons)
- **File Requirements**:
  - All files must end with a new line
  - First line must be `#!/usr/bin/node`
  - All files must be executable

## Setup

1. Install Node.js 14.x:
```bash
curl -sL https://deb.nodesource.com/setup_14.x | sudo -E bash -
sudo apt-get install -y nodejs
```

2. Install semistandard:
```bash
sudo npm install semistandard --global
```

## Usage

Make your JavaScript files executable:
```bash
chmod +x filename.js
```

Run a JavaScript file:
```bash
./filename.js
# or
node filename.js
```

Check code style:
```bash
semistandard filename.js
```
