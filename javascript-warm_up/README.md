# JavaScript - Warm up

This project contains introductory JavaScript scripts covering basic concepts like variables, data types, loops, functions, and more.

## Learning Objectives

- Why JavaScript programming is amazing
- How to run a JavaScript script
- How to create variables and constants
- Differences between var, const and let
- Data types available in JavaScript
- How to use control structures (if/else, loops)
- How to use functions
- Scope of variables
- How to manipulate objects and arrays
- How to import a file

## Requirements

- Ubuntu 20.04 LTS
- Node.js version 14.x
- Code must be semistandard compliant
- All files must be executable
- First line must be `#!/usr/bin/node`
