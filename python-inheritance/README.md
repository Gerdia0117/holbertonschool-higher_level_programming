# Python - Inheritance

![Project Badge](https://img.shields.io/badge/Python-Inheritance-blue)

## Description
This project explores the concept of inheritance in Python, covering topics such as single and multiple inheritance, method overriding, and the use of built-in functions like `isinstance()`, `issubclass()`, `type()`, and `super()`.

## Learning Objectives
By completing this project, you will understand:
- What superclass, baseclass, and parentclass are
- What a subclass is
- How to list all attributes and methods of a class or instance
- When an instance can have new attributes
- How to inherit a class from another
- How to define a class with multiple base classes
- The default class every class inherits from
- How to override a method or attribute
- Which attributes/methods are available by inheritance
- The purpose of inheritance
- How to use `isinstance`, `issubclass`, `type`, and `super` functions
